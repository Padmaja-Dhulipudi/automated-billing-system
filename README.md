# Automated Billing System

A full-stack MERN application for managing customers, products, and invoices. The system streamlines billing operations with secure authentication, inventory management, and invoice generation through a responsive web interface.

---

## Features

-  User Authentication (Login & Register)
-  Dashboard Overview
-  Customer Management
-  Product Management
-  Invoice Generation
-  Invoice History
-  Search & Filter
-  Responsive Design

---

##  Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JWT
- bcryptjs

##  Getting Started

### Clone the Repository

```bash
git clone https://github.com/Padmaja-Dhulipudi/automated-billing-system.git
cd automated-billing-system
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd ../server
npm install
npm run dev
```
